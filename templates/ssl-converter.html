PEM Format

Base64-encoded certificate with BEGIN/END headers. Most common format for web servers and applications.

  • Human-readable
  • Widely supported
  • Text-based
DER Format

Binary-encoded certificate format. More compact and efficient for storage and transmission.

  • Binary format
  • Compact size
  • Efficient storage
PKCS#7 Format

Cryptographic message syntax standard. Used for certificate chains and secure messaging.

  • Certificate chains
  • Secure messaging
  • Multiple certificates
PKCS#12 Format

Personal Information Exchange format. Contains both certificate and private key with password protection.

  • Password protected
  • Certificate + key
  • Secure storage
